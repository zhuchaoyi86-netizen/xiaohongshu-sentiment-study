# 第四部分 实证模型与情感影响分析

## 4.1 模型设定与变量定义

围绕“基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究”这一主题，本文将点赞热度作为核心被解释变量，并重点考察文本情感方向、情感强度以及正负情绪表达的不对称影响。考虑到小红书平台内容互动并不只受文本情绪影响，还会受到图文呈现方式和发布时间差异的共同作用，本文在模型中进一步加入配图数量、文本长度和发布时间时段等控制变量。

被解释变量设定为：

`log_likes = ln(likes + 1)`

其中，`likes` 表示笔记原始点赞数。采用对数化处理的原因在于原始点赞数据存在明显右偏分布，少数爆款笔记会显著拉高均值，因此需要通过对数变换降低极端值对估计结果的干扰。

核心解释变量主要包括三类。第一类是情感方向变量，包括连续型情感得分 `sentiment_score`，以及离散化后的 `positive_flag`、`negative_flag` 和 `neutral_flag`。其中，`sentiment_score` 由正向词频减去负向词频得到，数值越高表示文本整体情绪越积极。第二类是情感强度变量。本文根据情感得分将样本进一步划分为低强度正向、中强度正向和高强度正向三类，以考察不同情绪强弱是否会带来不同的点赞表现。第三类是不对称情绪变量，分别使用 `positive_word_count` 和 `negative_word_count` 衡量正向词与负向词在文本中的出现次数，从而识别二者在影响路径上是否存在差异。

控制变量包括：`text_length`，用于衡量文本内容丰富程度；`imagenumber`，用于衡量图片呈现数量；以及发布时间虚拟变量 `evening`、`afternoon` 和 `morning`，其中“凌晨”作为参照组。通过引入这些控制变量，本文能够更集中地识别情感变量对点赞热度的净影响。

## 4.2 模型设计

为从不同角度识别情感效应，本文构建了四组基础模型。

第一组模型用于检验情感得分的总体影响：

`log_likes = β0 + β1 * sentiment_score + β2 * text_length + β3 * imagenumber + β4 * evening + β5 * afternoon + β6 * morning + ε`

第二组模型用于检验正向与负向情绪的方向差异：

`log_likes = β0 + β1 * positive_flag + β2 * negative_flag + β3 * text_length + β4 * imagenumber + β5 * evening + β6 * afternoon + β7 * morning + ε`

第三组模型用于检验正向词与负向词的不对称影响：

`log_likes = β0 + β1 * positive_word_count + β2 * negative_word_count + β3 * text_length + β4 * imagenumber + β5 * evening + β6 * afternoon + β7 * morning + ε`

第四组模型用于比较不同情感强度的差异：

`log_likes = β0 + β1 * high_intensity + β2 * medium_intensity + β3 * low_positive + β4 * text_length + β5 * imagenumber + β6 * evening + β7 * afternoon + β8 * morning + ε`

其中，`high_intensity` 表示高强度正向情绪文本，`medium_intensity` 表示中强度正向情绪文本，`low_positive` 表示低强度正向情绪文本。

## 4.3 基础回归结果分析

从第一组模型结果看，`sentiment_score` 的回归系数为正，说明在控制文本长度、配图数量和发布时间后，文本情绪越积极，点赞热度整体上越高。虽然这一影响强度并不算特别大，但方向稳定且与研究预期一致。这表明在食品种草场景中，积极情绪表达的确有助于内容获得更多用户认同。

从第二组模型结果看，将连续型情感得分替换为简单的正负情绪虚拟变量后，正向和负向变量的系数方向都不再十分突出。这说明在食品种草笔记中，情绪是否“正向”本身并不足以完全解释点赞差异，情绪表达的程度、语气和文本组织方式可能比单纯的情绪标签更加重要。换句话说，简单地把文本粗分为“正向”与“非正向”，会损失部分有价值的信息。

从第三组模型结果看，正向词数量的系数为正，而负向词数量的系数为负，且负向词的绝对影响更强。这一结果说明，在食品种草场景中，积极评价能够带来一定的互动提升，但消极表达一旦出现，对点赞热度的抑制作用往往更明显。也就是说，正负情绪对点赞热度的影响并非完全对称，负面表述在互动形成中具有更强的“惩罚效应”。

从第四组模型结果看，不同情绪强度之间也存在一定差异。进一步结合分组统计结果可以发现，低强度和中强度正向情绪文本的平均点赞表现并不低于高强度正向文本，部分情况下甚至更高。这说明在食品种草笔记中，“适度积极”的表达可能比“过强情绪渲染”更容易获得用户接受。换言之，过度夸张、过度热烈的正向表达并不一定持续提升点赞热度，温和而真实的种草语气反而可能更有效。

此外，在四组模型中，`imagenumber` 的系数均保持稳定正向，且影响程度明显高于大多数情感变量，说明图文呈现方式仍然是影响点赞热度的重要因素。同时，发布时间变量也持续表现为正向影响，意味着与凌晨相比，上午、下午和晚上发布的食品种草笔记更容易获得点赞。由此可见，食品种草内容的热度形成机制是“文本情绪 + 图文形式 + 发布时间”共同作用的结果，而不仅仅由情感表达单独决定。

## 4.4 结论提炼

综合模型结果，本文可以得到以下几点结论。第一，食品种草笔记文本情感与点赞热度之间总体呈现正向关系，情绪越积极，内容越容易获得互动。第二，负向情绪词对点赞热度的抑制作用强于正向情绪词的促进作用，说明负面表达具有更明显的不对称影响。第三，情感强度并非越高越好，过强的正向渲染未必带来更高热度，适度积极、相对真实的表达可能更符合用户偏好。第四，图文呈现和发布时间对点赞表现具有持续而稳定的影响，因此在研究情感效应时有必要将其作为控制变量纳入模型。

这些结果表明，小红书食品种草笔记的互动热度并非由单一因素驱动，而是在情感内容、表达方式和内容呈现机制共同作用下形成的。对商家或内容创作者而言，单纯堆砌夸张正面词汇未必最有效，相比之下，兼顾真实感、适度情绪表达和较好的图文呈现方式，更可能提高内容的传播与互动效果。
