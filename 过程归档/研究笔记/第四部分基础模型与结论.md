# 第四部分基础模型与结论

## 一、模型设定

本文以食品种草笔记的点赞热度为研究对象，构建如下基础回归模型：

`log_likes = β0 + β1 * sentiment_score + β2 * text_length + β3 * imagenumber + β4 * evening + β5 * afternoon + β6 * morning + ε`

为了更直观地区分正向与负向情绪，本文进一步构建虚拟变量模型：

`log_likes = β0 + β1 * positive_flag + β2 * negative_flag + β3 * text_length + β4 * imagenumber + β5 * evening + β6 * afternoon + β7 * morning + ε`

其中，发布时间变量以“凌晨”为参照组。

## 二、变量定义

| 变量名 | 含义 | 说明 |
|---|---|---|
| `log_likes` | 点赞热度 | 因变量，采用 `ln(likes + 1)` |
| `sentiment_score` | 情感得分 | 正向词频减去负向词频 |
| `positive_flag` | 正向情绪变量 | 情感得分大于 0 记为 1 |
| `negative_flag` | 负向情绪变量 | 情感得分小于 0 记为 1 |
| `text_length` | 文本长度 | 笔记正文字符数 |
| `imagenumber` | 配图数量 | 笔记图片张数 |
| `evening` | 晚上发布 | 晚上发布记为 1 |
| `afternoon` | 下午发布 | 下午发布记为 1 |
| `morning` | 上午发布 | 上午发布记为 1 |

## 三、基础回归结果

### 模型 1：情感得分模型

模型 1 的结果显示，`sentiment_score` 的回归系数为 `0.0214`，方向为正，说明文本情感越积极，点赞热度整体上越有上升趋势。`imagenumber` 的系数为 `0.2757`，且正向影响最为明显，表明配图数量越多，笔记越容易获得较高点赞。`evening`、`afternoon` 和 `morning` 的系数均为正，说明相较于凌晨发布，其他时段发布的内容更容易获得互动。相比之下，`text_length` 的系数略为负，但影响较弱。模型的 `R²` 为 `0.0814`，说明基础模型能够解释一部分点赞差异，但仍有较多影响因素尚未纳入。

### 模型 2：正负情绪虚拟变量模型

模型 2 中，`positive_flag` 的系数为 `0.0648`，方向为正，说明正向情绪文本相较于非正向文本具有更高的点赞热度；`negative_flag` 的系数为 `-0.0309`，方向为负，说明负向情绪文本相较于非负向文本的点赞表现略低。与此同时，`imagenumber` 仍保持明显正向影响，系数为 `0.2771`；发布时间变量也继续表现为正向影响，说明发布时间是较稳定的辅助因素。模型的 `R²` 为 `0.0806`，与模型 1 基本接近。

## 四、简要结论

综合基础模型结果可以看出，文本情感与点赞热度之间存在一定正向关系，但其影响强度目前并不算特别大。相比之下，配图数量和发布时间对点赞热度的影响更为明显。这表明食品种草笔记的互动热度并非单纯由情绪表达决定，而是同时受到图文呈现方式和发布时间环境的共同影响。因此，在后续分析中，有必要将情感变量作为核心解释变量，并将图文和时间变量作为重要控制变量纳入模型。
