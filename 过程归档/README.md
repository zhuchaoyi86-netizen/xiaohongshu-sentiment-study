# 过程归档

这里存放的是研究形成过程中的材料，不是最终展示主线，但它能帮助读者理解这项研究是怎样一步步做出来的。

## 适合谁看

- 如果你想复盘研究推进过程，可以看这里。
- 如果你想修改论文或继续扩展分析，也可以从这里找线索。
- 如果你只是想快速了解研究结论，可以先不看这里，优先返回 [项目导读](/Users/xinxinhuashe/Documents/1/python课/项目导读.md) 和 [最终成果汇总](/Users/xinxinhuashe/Documents/1/python课/最终成果汇总)。

## 内容说明

- `研究笔记`：记录研究思路、阶段性写作和分析判断。
- `分析Notebook`：保留分析过程中使用的 Notebook。
- `中间文档`：存放阶段性生成的文档版本。

## 它在整个项目里的位置

可以把本目录理解为“幕后材料”：

- 想看研究结论，看 `最终成果汇总`
- 想看数据支撑，看 `最终成果汇总/03_数据结果`
- 想看研究是怎么逐步形成的，再回到这里
