# 基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究

本仓库用于整理 Python 课程项目《基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究》的研究材料、分析结果与最终汇报成果。

## 项目概述

本研究以小红书食品类种草笔记为对象，结合文本挖掘与基础计量分析方法，考察笔记情感特征是否会显著影响点赞热度。

研究工作主要包括：

- 数据清洗与样本筛选
- 文本情感量化与变量构建
- 描述性统计与可视化
- 基础回归与稳健性分析
- 论文与汇报材料整理

## 目录结构

- `最终成果汇总/`：适合直接查看和提交的整理后成果。
- `output/`：数据处理中间结果、分析产出和 Notebook 文件。
- `scripts/`：数据清洗、变量构建、回归分析和文档生成脚本。
- `notes/`：研究过程中的写作草稿、执行清单和分析笔记。
- `data/`：预留给原始数据或补充数据使用。
- `python/`：课程原始学习资料，默认不纳入本项目版本管理。
