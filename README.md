# 基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究

本项目的主线很简单：

我们从小红书原始评论/笔记数据出发，筛选出食品相关的种草文本，构建情感变量，再分析这些情感表达是否会影响点赞热度。

## 这个项目的核心问题

本研究围绕三个问题展开：

1. 小红书食品种草文本整体是偏正向还是偏负向？
2. 情绪表达越强、越集中，是否越容易获得高点赞？
3. 配图数量、发布时间等因素，会不会和情感变量一起影响热度？
