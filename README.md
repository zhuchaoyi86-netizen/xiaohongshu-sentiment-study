## 这个项目到底在研究什么

本项目的核心问题是：

食品相关的小红书种草文本中，情感表达越积极、越集中，是否越容易获得更高点赞热度？

围绕这个问题，项目大致经历了以下四步：

1. 从原始小红书数据出发，筛选出与食品主题相关、适合研究的文本样本。
2. 对文本进行清洗，构建情感得分、情绪方向、情绪密度等变量。
3. 对点赞热度进行描述性统计、可视化和基础回归分析。
4. 将分析结果整理为论文稿、PPT、图表和数据结果。
