# 小红书食品种草笔记情感与点赞热度研究

本研究最初为学校 Python 课程结课作业，并在此基础上持续修改完善。

本项目以 **9999 条小红书商品笔记原始数据** 为起点，经过文本清洗和食品种草样本筛选，得到 **2899 条食品种草笔记**。研究进一步从笔记正文中提取情感方向、情绪密度和混合评价等特征，分析它们与点赞热度之间的关系。

> 研究主线：原始笔记数据 -> 数据清洗 -> 食品种草样本筛选 -> 情感变量构建 -> 点赞热度分析 -> 论文与汇报成果

建议先打开 [一页研究摘要](./研究摘要.md)。它用一页内容说明研究问题、数据、方法、主要发现和最终成果。

## 主要结果

- 原始 9999 条商品笔记经清洗后保留 9692 条，其中 3320 条属于食品类，最终识别出 2899 条食品种草笔记。
- 描述性统计显示，情绪密度较高的笔记平均点赞更多，但加入控制变量和稳健标准误后，该关系未达到常用显著性水平。
- 混合评价与点赞热度呈稳定的显著负相关，在多种稳健性检验中结论保持一致。
- 配图数量与点赞热度显著正相关，说明内容呈现方式也是影响互动的重要因素。

上述结果表示变量之间的统计关联，不能直接解释为因果关系。
