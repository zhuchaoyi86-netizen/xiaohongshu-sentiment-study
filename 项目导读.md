# 项目导读

这份导读的目的不是介绍文件夹，而是带你按研究顺序看懂这个项目。

## 1. 研究在做什么

这项研究关注的是：

小红书食品种草文本中的情感表达，是否会影响内容的点赞热度。

这里的“情感表达”不是笼统地说一句“正面情绪多”，而是进一步拆成了几个可分析的问题：

- 文本整体更偏正向还是负向？
- 情绪表达越集中，点赞会不会更高？
- 同时包含正向和负向评价的“混合评价”，会不会压低点赞？

如果你想先快速看研究结果，可以直接打开：

- [最终汇报 PPT](./最终成果汇总/02_PPT最终版/基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究_最终版.pptx)
- [论文稿](./最终成果汇总/01_论文与说明/基于文本挖掘的小红书食品种草笔记情感对点赞热度的影响研究_论文稿.docx)

## 2. 数据从哪里来

这项研究不是直接拿“食品种草数据”开始分析的，而是从更原始的小红书商品/评论数据出发，逐步筛选得到最终样本。

整个数据脉络可以概括为：

原始平台数据  
-> 清洗后的商品/笔记数据  
-> 食品相关样本  
-> 食品种草文本样本  
-> 情感分析与模型分析结果

如果你想看这条线在文件里是怎样落下来的，建议依次打开：

- [商品笔记清洗汇总](./最终成果汇总/03_数据结果/02_筛选与清洗说明/小红书商品笔记_清洗汇总.csv)
- [食品种草样本筛选汇总](./最终成果汇总/03_数据结果/02_筛选与清洗说明/小红书食品种草笔记_筛选汇总.csv)

## 3. 研究是怎样推进的

### 第一步：先把研究样本筛出来

研究首先要解决“哪些文本才算本研究真正关心的样本”。

因此，项目先做了样本筛选和清洗，把无关、重复或低质量文本排除掉，再保留食品主题和种草表达较明确的文本。

对应文件：

- [低质量样本示例](./最终成果汇总/03_数据结果/02_筛选与清洗说明/小红书商品笔记_低质量样本示例.csv)
- [商品笔记清洗汇总](./最终成果汇总/03_数据结果/02_筛选与清洗说明/小红书商品笔记_清洗汇总.csv)
- [食品种草研究优化汇总](./最终成果汇总/03_数据结果/02_筛选与清洗说明/小红书食品种草笔记_研究优化汇总.csv)

### 第二步：把文本变成可以分析的变量

样本筛出来之后，研究进一步把文本信息转成结构化变量，比如：

- 情感方向
- 情绪强度
- 情绪密度
- 混合评价
- 发布时间
- 配图数量
