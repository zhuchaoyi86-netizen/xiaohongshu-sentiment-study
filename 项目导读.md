# 项目导读

## 1. 研究主题

本项目研究的是：

小红书食品种草文本中的情感表达，是否会影响内容的点赞热度。

更具体地说，我们不是只想知道“正面情绪多不多”，而是想回答：

- 食品种草文本整体是偏正向还是偏负向？
- 情绪表达越强烈、越集中，是否更容易带来高点赞？
- 配图数量、发布时间等因素，会不会和情感变量一起影响热度？

## 2. 数据从哪里来

本研究从小红书原始评论/笔记数据出发，先保留可用于文本分析和热度研究的字段，再逐步筛选出与食品主题、种草表达相关的样本。

因此，整个研究的数据脉络是：

原始平台数据
-> 清洗后的商品/笔记数据
-> 食品相关样本
-> 食品种草文本样本
-> 情感分析与模型分析结果
