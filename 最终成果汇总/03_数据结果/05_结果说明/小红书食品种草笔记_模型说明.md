# 基础模型说明

本文的基础回归模型用于检验文本情感特征与点赞热度之间的关系，因变量为点赞热度的对数形式 `log_likes`。

## 模型 1：总体情感得分模型

`log_likes = b0 + b1*sentiment_score + b2*text_length + b3*imagenumber + b4*evening + b5*afternoon + b6*morning + e`

模型 1 用于检验综合情感得分对点赞热度的总体影响。

## 模型 2：情感方向虚拟变量模型

`log_likes = b0 + b1*positive_flag + b2*negative_flag + b3*text_length + b4*imagenumber + b5*evening + b6*afternoon + b7*morning + e`

模型 2 用于比较正向情感与负向情感对点赞热度的方向性差异。

## 控制变量说明

所有基础模型均控制以下变量：

- `text_length`：笔记文本长度
- `imagenumber`：配图数量
- `morning`、`afternoon`、`evening`：发布时间时段虚拟变量

其中，发布时间虚拟变量以“凌晨”时段为参照组。
