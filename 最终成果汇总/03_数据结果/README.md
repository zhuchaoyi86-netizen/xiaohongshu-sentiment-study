# 数据结果说明

这一部分不是“附件区”，而是整项研究的数据主干。

如果你想知道这项关于“小红书食品种草文本情感与点赞热度”的研究，是怎样从原始数据一步步走到结论的，就从这里看。

## 推荐阅读顺序

### 第一步：先看样本怎么筛出来

先看：

- [商品笔记清洗汇总](./02_筛选与清洗说明/小红书商品笔记_清洗汇总.csv)
- [食品种草样本筛选汇总](./02_筛选与清洗说明/小红书食品种草笔记_筛选汇总.csv)
- [研究优化汇总](./02_筛选与清洗说明/小红书食品种草笔记_研究优化汇总.csv)

这一步主要回答：原始平台数据为什么最后会变成“食品种草文本样本”。

### 第二步：再看最终分析用的数据

然后看：

- [清洗后的食品种草数据](./01_核心数据/小红书食品种草笔记_清洗后数据.csv)
- [研究优化数据](./01_核心数据/小红书食品种草笔记_研究优化数据.csv)
- [情感分析数据](./01_核心数据/小红书食品种草笔记_情感分析数据.csv)

这一步主要回答：真正进入分析流程的数据底稿到底是什么。

### 第三步：看变量是怎么表现出来的

接着看：

- [情感方向统计](./03_变量统计结果/小红书食品种草笔记_变量分析_情感方向.csv)
- [情绪密度统计](./03_变量统计结果/小红书食品种草笔记_变量分析_情绪密度.csv)
- [情绪强度统计](./03_变量统计结果/小红书食品种草笔记_变量分析_情绪强度.csv)
- [混合评价统计](./03_变量统计结果/小红书食品种草笔记_变量分析_混合评价.csv)
- [发布时间统计](./03_变量统计结果/小红书食品种草笔记_变量分析_发布时间.csv)

这一步主要回答：研究里说的“情感方向、情绪密度、发布时间”等变量，具体在数据里长什么样。

### 第四步：最后看模型结果

最后看：

- [基础回归结果](./04_模型结果/小红书食品种草笔记_基础回归结果.csv)
- [情感模型结果](./04_模型结果/小红书食品种草笔记_情感模型结果.csv)
- [基准回归关键结果](./04_模型结果/第四部分_基准回归关键结果.csv)
- [鲁棒性结果：情绪密度](./04_模型结果/第四部分_鲁棒性_情绪密度.csv)
- [鲁棒性结果：混合评价](./04_模型结果/第四部分_鲁棒性_混合评价.csv)

这一步主要回答：情感变量和点赞热度之间到底有没有关系，这种关系是否稳定。
