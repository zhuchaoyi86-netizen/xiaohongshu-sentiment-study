# 数据结果说明

这一部分回答的核心问题是：

这项关于“小红书食品种草文本情感与点赞热度”的研究，具体是如何从原始数据一步步走到分析结果的？

## 这部分怎么读

建议按照下面顺序阅读。

### 1. 先看样本是怎么来的

- `02_筛选与清洗说明`

这一部分展示从原始数据到研究样本的筛选与清洗过程，包括低质量样本示例、清洗汇总和研究优化汇总。

### 2. 再看最终用了哪些核心数据

- `01_核心数据`

这一部分包含研究真正使用的关键数据文件，包括清洗后数据、食品种草样本数据和情感分析数据。

### 3. 然后看变量是怎么构建和描述的

- `03_变量统计结果`
- `05_结果说明`

这里可以看到情感方向、情绪密度、情绪强度、发布时间、配图数量等变量的统计结果，以及模型和变量的文字说明。

### 4. 最后看模型结果

- `04_模型结果`

这里集中放置基础回归、稳健性检验、模型概况和关键结果，是支撑研究结论的核心结果区。

## 五个子目录分别在做什么

- `01_核心数据`：最终分析直接使用的数据底稿。
- `02_筛选与清洗说明`：说明研究对象如何从原始数据中被筛选出来。
- `03_变量统计结果`：说明情感变量、时间变量和图文变量的分布特征。
- `04_模型结果`：说明情感变量与点赞热度之间的统计关系。
- `05_结果说明`：帮助读者快速理解变量和模型的含义。

## 如果只想看最关键的支撑材料

建议优先打开：

1. `02_筛选与清洗说明`
2. `03_变量统计结果`
3. `04_模型结果`
