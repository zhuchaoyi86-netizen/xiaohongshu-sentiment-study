# 第三部分 数据预处理与变量构建

## 3.1 样本筛选与原始数据清洗

本文研究主题为“小红书食品种草笔记情感对点赞热度的影响”。因此，在原始小红书商品笔记数据基础上，首先围绕研究主题对样本进行筛选与清洗。具体而言，本文先保留与研究问题直接相关的核心字段，包括笔记标题、正文内容、点赞数、发布时间、配图数量、商品类别等信息；随后仅保留商品类别为“食品”的样本，以确保研究对象与“食品种草”情境保持一致。在此基础上，进一步结合“种草”“推荐”“安利”“回购”“分享”“合集”“测评”“好吃”等典型表达，对具有明显消费推荐特征的食品种草笔记进行识别，从而形成研究所需的目标样本。

完成样本筛选后，本文继续对文本数据进行基础清洗处理。首先，对正文缺失、无法识别或无有效语义内容的样本予以剔除。其次，依据样本唯一标识对重复记录进行核查，避免同一条笔记被重复计入分析。再次，对极短文本、仅由符号构成的文本、纯语气词文本以及语义信息明显不足的低质量文本进行识别与删除，以提升后续文本分析结果的有效性。与此同时，针对原始文本中存在的乱码字符、异常空白和格式不统一等问题，本文进行了标准化处理。进一步地，考虑到平台文本中常含有活动宣传语、价格罗列、优惠提示和模板化营销表达，本文还对营销噪声较强、情绪信息过弱的样本进行了额外筛除，以保证保留下来的文本能够更好地服务于情感分析任务。

在上述处理基础上，本文最终保留清洗后有效样本 9692 条。在此范围内，进一步筛选得到食品类样本 3320 条，其中符合食品种草特征的样本 2899 条，占食品类样本的 87.32%。从样本结构看，该数据集较好覆盖了食品推荐、消费体验分享、口味评价和购买建议等典型内容场景，为后续开展情感分析和点赞热度研究提供了较为可靠的数据基础。

## 3.2 文本清洗与情感量化处理

为了更准确地识别食品种草笔记中的情绪表达，本文在基础清洗之上进一步进行了面向情感分析的文本预处理。考虑到小红书平台文本具有明显的社交媒体特征，原始正文中往往同时包含话题标签、用户提及、平台表情标记、价格数字、营销提示语和其他与情绪识别无关的信息。如果直接将原始文本用于情感分析，可能会干扰情感判断并降低文本变量的解释性。因此，本文在保留原始清洗文本的同时，额外构建了专门用于情感识别的文本字段。

在情感分析文本构建过程中，本文主要进行了以下处理：第一，删除或弱化 `#话题标签`、`@用户提及`、平台表情标记、链接和无关长数字等平台噪声信息；第二，对重复标点、异常符号和明显格式干扰项进行标准化；第三，在不破坏原始语义的前提下，尽量保留能够体现文本情绪倾向的描述词、评价词和态度词。经过上述处理后，得到的情感分析文本能够更加集中地反映作者对食品商品或消费体验的主观态度。

在情感量化方法上，本文采用基于情感词识别的量化方式，将文本信息转化为可进入计量模型的结构化变量。具体做法是预先设定正向情绪词和负向情绪词，对每条笔记中的相关词语进行统计，并构造情感得分变量：

`sentiment_score = 正向词频 - 负向词频`

其中，正向情绪词主要包括“好吃”“喜欢”“推荐”“安利”“回购”“宝藏”“惊艳”“满意”“真香”“绝了”等，负向情绪词主要包括“难吃”“一般”“失望”“踩雷”“避雷”“不推荐”“翻车”“后悔”等。若情感得分大于 0，则判定该文本以正向情绪为主；若情感得分小于 0，则判定该文本以负向情绪为主；若情感得分等于 0，则视为中性或情绪表达不明显文本。基于这一规则，本文进一步构建了 `positive_flag`、`negative_flag` 和 `neutral_flag` 等虚拟变量，以便在后续模型中从不同角度考察情感表达的影响。

从样本分布来看，食品种草笔记整体呈现明显的正向表达特征。情感识别结果显示，正向情绪样本 2403 条，占比约 82.89%；负向情绪样本 123 条，占比约 4.24%；中性样本 373 条，占比约 12.87%。这一分布与食品种草笔记以推荐、分享和消费体验展示为主的内容属性基本一致，也说明本文构建的情感变量与研究场景具有较好的匹配性。

## 3.3 情感变量与衍生变量构建

围绕“食品种草笔记情感对点赞热度的影响”这一研究主题，本文在完成文本清洗后，重点构建了三类情感变量。

第一类是情感方向变量。本文使用 `sentiment_score` 衡量笔记文本的整体情感倾向，其计算方式为正向词频减去负向词频。该变量能够反映文本总体是更偏积极表达还是更偏消极表达。在此基础上，进一步构建 `positive_flag`、`negative_flag` 和 `neutral_flag` 三个虚拟变量，分别用于识别正向情绪文本、负向情绪文本和中性文本。通过这组变量，后续研究可以从“情感方向差异”的角度考察不同情绪类型对点赞热度的影响。

第二类是情感强度变量。考虑到社交媒体文本中的情绪表达不仅有方向差异，还存在强弱差异，本文进一步根据情感得分的大小划分文本的情绪强度水平。具体而言，可将样本区分为低强度正向、中强度正向和高强度正向等不同层次，用于刻画“温和表达”与“强烈表达”之间的差异。该变量设计有助于后续进一步讨论：食品种草笔记中的情绪表达是否越强烈越容易获得点赞，还是适度积极、相对真实的表达更受用户欢迎。

第三类是情感结构变量。为更细致地观察正负情绪表达在文本中的作用方式，本文分别统计正向情绪词出现次数和负向情绪词出现次数，构造 `positive_word_count` 与 `negative_word_count` 两个变量。与单一情感得分相比，这组变量能够帮助识别正向与负向表达是否存在不对称影响。例如，积极评价可能对点赞具有促进作用，而消极评价可能对点赞具有更强的抑制作用。通过这种变量设计，本文能够从多个角度挖掘情感表达的影响机制。

除核心情感变量外，本文还进一步构建了若干辅助变量。首先，构建文本长度变量 `text_length`，用于衡量笔记内容的信息丰富程度。通常而言，文本越长，所承载的产品信息、消费体验和情绪表达越充分，因此可能对用户互动产生影响。其次，构建配图数量变量 `imagenumber`，并进一步转化为是否配图变量 `has_image`。考虑到小红书是典型的图文平台，图片呈现方式可能显著影响用户的注意力与点赞行为，因此该变量具有较强的解释意义。

再次，本文根据发布时间构建时间时段变量，将样本划分为凌晨、上午、下午和晚上四个区间，以刻画不同发布时间对互动热度的影响。由于平台用户活跃程度在不同时段可能存在明显差异，因此将时间变量纳入分析有助于提高后续研究设计的合理性。此外，本文还保留了话题数量、感叹号数量、问号数量以及营销话术标记等文本形式变量，用于从表达强度、文本风格和平台话语特征角度补充刻画笔记内容。

综合来看，本文最终形成了一套以情感变量为核心、以文本特征、图文特征和时间特征为辅助的变量体系。其中，情感方向变量用于刻画文本态度倾向，情感强度变量用于识别不同强弱的情绪表达，情感结构变量用于观察正负情绪词的作用差异，而文本长度、配图相关变量和时间时段变量则作为辅助解释因素。这一变量构建框架既能够突出研究主题，也为后续从多个角度探索“情感如何影响点赞热度”提供了清晰的分析基础。

## 3.4 点赞热度变量处理与数据优化

本文将点赞数作为衡量笔记互动热度的核心结果变量。描述性统计结果显示，点赞数在样本中呈现出明显的右偏分布，即大部分笔记的点赞量相对有限，而少数高热度笔记获得了远高于平均水平的点赞数。这种分布特征容易导致极端值对模型估计结果产生较大影响，不利于后续回归分析的稳定性与解释性。

为减弱异常高值带来的影响，本文首先对点赞数进行对数变换处理，构造因变量 `log_likes = ln(likes + 1)`。其中，加 1 的原因在于原始点赞数中可能存在 0 值，直接取对数会出现无定义问题。经过对数变换后，点赞变量的分布更为平缓，有利于提升模型估计的稳定性，也更符合常见的计量分析处理方式。

在此基础上，本文进一步对点赞热度变量进行了去极值优化处理。考虑到点赞数分布存在明显长尾特征，少数爆款笔记可能对整体估计结果产生过强影响，因此本文采用分位数缩尾的方法，对点赞数进行 Winsorize 处理，并构造 `likes_winsorized` 和 `log_likes_winsorized` 两个优化变量。同时，本文还设置 `high_like_outlier_flag` 标记高点赞离群样本，以便后续在稳健性检验中识别高热度极端观测值的影响。这样既保留了高热度内容的信息，又在一定程度上降低了极端值对整体模型的干扰。

除点赞热度变量外，本文还对文本长度和情感变量进行了进一步优化。对于文本长度，本文同样采用分位数缩尾思路，构造 `text_length_winsorized`，并设置 `high_text_outlier_flag` 标记超长文本样本，以避免个别超长笔记对文本长度变量的估计结果造成过度拉动。对于情感变量，除原有情感得分外，本文进一步构造 `sentiment_score_winsorized` 和 `sentiment_abs`，前者用于减弱极端情绪值影响，后者用于刻画情绪强度本身而不区分方向。

为了更细致地刻画情绪表达特征，本文还新增了若干研究优化变量。首先，构造 `sentiment_density`，即单位文本长度中的情绪强度水平，用于衡量文本在相同篇幅下的情绪表达密度。其次，构造 `positive_ratio` 与 `negative_ratio`，分别表示正向词和负向词在全部情绪词中的占比，从而更加直观地反映文本内部的情感结构。再次，构造 `emotion_word_total` 变量，用于衡量文本中情绪表达的总体活跃程度。进一步地，本文依据情感得分大小构造 `sentiment_intensity_level`，将样本划分为低强度正向、中强度正向、高强度正向、负向和中性等层级，以便后续从情绪强弱角度进行比较分析。

考虑到食品种草笔记中经常同时存在“推荐/种草”与“避雷/踩雷”并存的复杂表达，本文还额外构造 `mixed_review_flag` 变量，用于识别混合评价型文本。这一变量能够帮助后续研究进一步区分“单一正向种草”“单一负向评价”和“正负并存评价”三类不同内容形态，从而更细致地观察复杂情绪结构对点赞热度的影响。

总体来看，经过样本筛选、文本清洗、情感量化、变量构建以及多层次数据优化处理后，原始小红书食品种草笔记数据已被转化为一套适用于文本挖掘与后续实证分析的结构化研究数据。至此，第三部分完成了从原始平台文本到研究变量体系的转换，并在基础变量之外进一步形成了情绪方向、情绪强度、情绪密度与混合评价等扩展指标，为后续更深入地分析“情感如何影响点赞热度”做好了准备。

## 3.5 基于构建变量的初步数据分析

在完成变量构建与数据优化后，本文进一步基于核心变量进行了初步描述性分析，以观察不同情绪特征下点赞热度的分布差异。首先，从情感方向变量来看，`positive_flag`、`negative_flag` 和 `neutral_flag` 对应的样本在点赞表现上存在明显差异。其中，正向情绪文本的平均点赞表现整体高于负向情绪文本，而中性文本处于两者之间。这说明在食品种草场景下，积极表达更容易获得用户认同，而消极评价更难形成较高互动。

其次，从情感强度变量来看，`sentiment_intensity_level` 将样本划分为负向、中性、低强度正向、中强度正向和高强度正向等层级。初步统计结果表明，正向样本内部不同强度层级之间也存在差异，而且这种差异并不表现为“情绪越强、点赞越高”的简单线性关系。换言之，强烈正向渲染并不必然带来最高点赞表现，适度积极、表达相对自然的种草文本同样可能获得较好的互动效果。

再次，从情绪密度变量来看，`sentiment_density` 越高的样本，其平均点赞表现越好。这说明在食品种草笔记中，真正影响互动热度的可能不仅是情绪词出现的绝对数量，还包括情绪表达在文本中的集中程度。与冗长但情绪分散的表达相比，情绪更集中、更鲜明的内容更容易吸引用户注意并引发互动。因此，“情感倾向更强的帖子点赞数量更多”这一判断需要更精细地表述为：在控制文本长度影响后，情绪表达更集中、更鲜明的帖子整体上更容易获得较高点赞热度。

从情感结构变量来看，`positive_word_count`、`negative_word_count`、`positive_ratio` 和 `negative_ratio` 共同反映了文本内部的情绪构成。初步比较发现，正向情绪占比更高的文本整体互动表现更好，说明用户更容易对情绪结构较为明确、评价倾向较为清晰的种草内容产生认同。进一步观察发现，混合评价型文本同样具有研究价值。本文通过 `mixed_review_flag` 识别出一部分同时包含“种草/推荐”和“避雷/踩雷”等表达的样本，这类文本通常兼具推荐与提醒属性。初步统计发现，混合评价文本的平均点赞表现低于单一评价文本，说明用户在面对正负信息并存的内容时，互动反应可能更加谨慎。这一现象提示我们，在食品种草语境中，复杂评价结构可能改变用户对内容可信度和传播价值的判断方式。

从点赞热度变量本身来看，原始点赞数 `likes` 具有明显长尾特征，少数爆款笔记显著拉高整体均值，因此本文在保留原始点赞信息的同时，进一步使用 `log_likes`、`likes_winsorized` 和 `log_likes_winsorized` 作为优化后的热度指标。初步比较表明，缩尾后的点赞变量能够在保留热度差异的同时减弱极端值干扰，更适合用于后续稳健分析。

最后，从辅助变量的角度看，`imagenumber`、`has_image` 与 `time_period` 也表现出稳定的区分作用。由于本样本几乎均为图文笔记，因此 `has_image` 的区分度有限，但 `imagenumber` 仍然显示出明显差异：图片数量较少的样本平均点赞偏低，而多图内容的互动表现相对更好。与此同时，相较于凌晨发布，上午、下午和晚上发布的内容整体更容易获得点赞，说明发布时间环境仍然会影响互动表现。这表明在小红书平台中，文本情绪虽然是研究核心，但其影响效果仍然需要放在图文呈现方式和发布时间环境中共同考察。总体而言，这些初步数据分析结果说明，本文在 3.3 和 3.4 中构建的情感方向、情感强度、情绪密度、情绪结构、点赞热度优化变量以及辅助变量都具有较好的区分能力，也为后续开展更深入的实证分析提供了直观依据。
