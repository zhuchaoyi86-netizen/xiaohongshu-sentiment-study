# 最终成果汇总

本文件夹集中存放本项目当前阶段可直接提交、展示或分享的核心成果，便于课程汇报、论文整合和项目归档。

## 内容概览

- `01_论文与说明`：论文稿、报告文档及补充版说明材料。
- `02_PPT最终版`：当前用于汇报展示的最终版 PPT。
- `03_数据结果`：清洗结果、变量分析表、基础回归结果及模型说明。
- `04_图表`：可直接用于论文、报告或 PPT 的核心图表。
- `05_过程说明与笔记`：研究过程中的正式成稿、结论说明与分析笔记。

## 建议使用方式

- 需要交作业或打包发送时，优先从本文件夹选取材料。
- 需要继续修改分析内容时，请回到项目根目录中的 `scripts/`、`output/` 和 `notes/` 进行更新。
- 完成修改后，再将最终版本同步整理到本文件夹中。

## 说明

本文件夹强调“成果整合”和“直接可用”，因此仅保留当前阶段最重要的交付文件，不再保留大量中间迭代视觉稿与拆分素材。
