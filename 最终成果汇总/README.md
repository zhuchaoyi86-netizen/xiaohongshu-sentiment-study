# 最终成果汇总

本文件夹用于集中存放“小红书食品种草笔记情感对点赞热度的影响研究”的当前可交付成果。

## 文件夹说明

- `01_论文与说明`：论文稿、报告文档和补充版 docx。
- `02_PPT最终版`：当前最终汇报 PPT。
- `03_数据结果`：清洗后的数据、变量分析结果、回归结果和模型说明。
- `04_图表`：报告与 PPT 可直接使用的图表图片。
- `05_过程说明与笔记`：研究过程中的正式成稿、说明和结论笔记。

## 当前整理原则

- 保留最终成果与核心分析结果。
- 将明显重复的视觉稿迭代文件夹从项目根目录移除。
- 脚本、原始课程资料、数据处理代码继续保留在原项目中，便于后续修改。
