# 第四部分基础模型与结论

## 1. 模型设定

本文以食品种草笔记的点赞热度为研究对象，构建以下基础回归模型：

`log_likes = β0 + β1 * sentiment_score + β2 * text_length + β3 * imagenumber + β4 * evening + β5 * afternoon + β6 * morning + ε`

此外，为了更直观地区分正向与负向情绪，进一步构建虚拟变量模型：

`log_likes = β0 + β1 * positive_flag + β2 * negative_flag + β3 * text_length + β4 * imagenumber + β5 * evening + β6 * afternoon + β7 * morning + ε`

其中，时间变量以“凌晨”为参照组。

## 2. 变量定义

| 变量名 | 含义 | 说明 |
|---|---|---|
| `log_likes` | 点赞热度 | 因变量，使用 `ln(likes + 1)` |
| `sentiment_score` | 情感得分 | 正向词频减去负向词频 |
| `positive_flag` | 正向情绪变量 | 情感得分大于 0 记为 1 |
| `negative_flag` | 负向情绪变量 | 情感得分小于 0 记为 1 |
| `text_length` | 文本长度 | 笔记正文字符数 |
| `imagenumber` | 配图数量 | 笔记图片张数 |
| `evening` | 晚上发布 | 晚上发布记为 1 |
| `afternoon` | 下午发布 | 下午发布记为 1 |
| `morning` | 上午发布 | 上午发布记为 1 |

## 3. 基础回归结果

### 模型 1：情感得分模型

- `sentiment_score` 系数为 `0.0214`，为正，说明情感越积极，点赞热度整体上有上升趋势。
- `imagenumber` 系数为 `0.2757`，为正且影响最明显，说明配图数量越多，点赞热度越高。
- `evening`、`afternoon`、`morning` 系数均为正，说明相较于凌晨发布，其他时段的内容更容易获得点赞。
- `text_length` 系数略为负，但影响较弱。
- 模型 `R²` 为 `0.0814`，说明该基础模型能够解释一部分点赞差异，但仍有不少影响因素未被纳入。

### 模型 2：正负情绪虚拟变量模型

- `positive_flag` 系数为 `0.0648`，方向为正，说明正向情绪文本相较于非正向文本，点赞热度略高。
- `negative_flag` 系数为 `-0.0309`，方向为负，说明负向情绪文本相较于非负向文本，点赞热度略低。
- `imagenumber` 仍保持明显正向影响，系数为 `0.2771`。
- 发布时间变量依然表现为正向影响，说明发布时间是较稳定的辅助因素。
- 模型 `R²` 为 `0.0806`，与模型 1 接近。

## 4. 简要结论

- 从基础模型看，文本情感与点赞热度之间存在一定正向关系，但影响强度目前不算特别大。
- 相比之下，配图数量和发布时间对点赞热度的影响更明显。
- 这说明食品种草笔记的互动热度并不只由情绪表达决定，还会受到图文呈现方式和发布时间共同影响。
- 因此，后续分析可以将情感变量作为核心解释变量，同时将图文和时间变量作为控制变量。
